# My HTML CV

This project is a personal online curriculum vitae (CV) for Gabriel García Garrido, meticulously crafted using pure HTML, CSS, and JavaScript. It serves as a comprehensive showcase of professional background, technical skills, education, and experience, demonstrating a strong command of front-end web development without relying on external frameworks.

## Features

*   **Responsive Design:** Optimized for various screen sizes to ensure a consistent experience across devices.
*   **Interactive Sections:** Navigable sections for "About Me," "Education," "Skills," "Experience," "Projects," and "Contact."
*   **Dynamic Skill Display:** Skills are categorized and presented with progress bars, offering a clear overview of proficiencies in different technologies (Programming Languages, Databases, Back-End, Front-End, AI Tools, Infrastructure, Containers, DevOps, Cloud).
*   **PDF Generation:** Includes functionality to generate a professional PDF version of the CV directly from the web page using `html2pdf.js`.
*   **Visual Enhancements:** Features a dynamic "Matrix" style background on the hero section and subtle fade-in animations for content.
*   **Social and Project Links:** Direct links to photography portfolio (Instagram) and GitHub profile for further exploration.

## Technologies Used

*   **HTML5:** For structuring the content.
*   **CSS3:** For styling and layout, including custom animations and responsive design.
*   **JavaScript (Vanilla JS & jQuery):** For interactive elements, dynamic content loading, navigation toggles, and PDF generation.
*   **html2pdf.js:** A client-side library for converting HTML content to PDF.
*   **Google Fonts:** For custom typography.
*   **Font Awesome & Material Icons:** For professional iconography.

## Project Structure

The project follows a straightforward structure:

```
my-html-cv/
├── css/             # Contains custom CSS styles
│   └── styles.css
├── images/          # Stores all images used in the CV (profile photo, logos, etc.)

│   ├── javascript.js
│   └── matrix.js

```

## How to Run Locally

To run this project on your local machine, simply clone the repository and open the `index.html` file in your web browser.

```bash
git clone https://github.com/Gaboxondo/my-html-cv.git
cd my-html-cv
start index.html # On Windows
# open index.html # On macOS
# xdg-open index.html # On Linux
```
